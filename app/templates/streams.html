How to set up in OBS

  1. Open OBS and go to Settings → Stream
  2. Set Service to Custom...
  3. Paste the Server URL above into the Server field
  4. Set Stream Key to any name you want for this stream (e.g. my-stream)
  5. Click OK, then hit Start Streaming

The stream page will be live at /live/<your-stream-key>/ once you connect. Reuse the same key to resume a previous stream.
